# Anime Academy

#### Stack:
- FastAPI
- Postgres
- Redis
- ReactJS
- ViteJS
- SCSS

#### Backend architecture

```yaml
├── src
    ├── clients/
    ├── server/
      ├── routers/
      ├── app.py
      ├── handle_errors.py
    ├── services/
    ├── settings/
    ├── container.py
    ├── __main__.py
```
